# 새마을금고 업무방법서 조회 & 학습 앱

새마을금고 **수신업무방법서**와 **여신업무방법서**를 하나의 앱에서 조회하고, 신입직원 필수 내용을 OX 퀴즈로 학습할 수 있는 단일 파일 웹앱입니다.

---

## 수록 문서 및 통계

| 항목 | 수신업무방법서 | 여신업무방법서 |
|------|--------------|--------------|
| 편(篇) | 2편 | 6편 |
| 장(章) | 11장 | 39장 |
| 조문 수 | 909개 | 1,200개 |
| OX 학습 문항 | 50문항 | 50문항 |

## 주요 기능

### 📖 조문 조회
- **문서 전환 토글**: 헤더에서 수신/여신 방법서를 탭 하나로 전환
- **키워드 검색**: 조문번호·제목·본문을 즉시 검색, 일치 부분 강조
- **목차 탐색**: 편 → 장 → 절 → 조문 계층으로 범주화 탐색
- **자주 찾는 키워드 칩**: 문서별 맞춤 키워드 빠른 검색
- **개정이력 배지**: 개정(주황)·신설(초록)·삭제(빨강) 색상 구분
- **부칙 보기**: 개정 연혁 전체 조회

### ✅ OX 학습
- **신입직원 필수 100문항** (수신 50 + 여신 50)
- **분야 선택 출제**: 원하는 카테고리만 선택해 풀기
- **문항별 즉시 피드백**: 선택 즉시 정답/오답 + 해설 + 관련 조항 연결
- **정답이어도 관련 조항 항상 출력**: 학습 효과 극대화
- **점수·등급 제공**: 우수(90%↑) / 보통(60%↑) / 보완 필요
- **오답 조문 모아보기**: 틀린 문항의 관련 조문 한 화면에 복습

### 📱 모바일 최적화
- **하단 탭바**: 조문 조회 / OX 학습 전환
- **히어로 화면**: 수신(파란색) / 여신(초록색) 문서별 색상 구분
- **Android 하드웨어 뒤로가기 / iOS 스와이프 뒤로가기 완전 지원**
  - History API 기반 라우터로 화면 단계별 복원
  - 문서 전환 후 뒤로가기 시 이전 문서 화면으로 정확히 복원
- **iOS 입력 자동확대 방지**: 검색 입력 16px 고정
- **safe-area(노치·홈바) 대응**: 하단 탭바 환경 자동 조정
- **오프라인 동작**: 폰트 로드 실패 시 시스템 한국어 폰트 자동 폴백

## 사용 방법

`index.html`(또는 `새마을금고_업무방법서_조회.html`)을 브라우저에서 열면 됩니다.  
별도 설치·서버·인터넷 연결이 필요 없습니다.

### GitHub Pages로 배포하기

1. 이 저장소를 GitHub에 올립니다.
2. 저장소 **Settings → Pages → Branch: main / root** 선택 후 저장합니다.
3. 발급된 주소(`https://<사용자명>.github.io/<저장소명>/`)로 접속합니다.

> 📱 휴대폰 브라우저에서 접속 후 **홈 화면에 추가**하면 실제 앱처럼 사용할 수 있습니다.

## 기술 메모

- **단일 파일**: 수신 909개 + 여신 1,200개 = 총 2,109개 조문과 100개 OX 문항이 HTML 내부에 포함
- **외부 의존성**: Pretendard 폰트 CDN 1개. 로드 실패 시 Apple SD Gothic Neo → 맑은 고딕 → 시스템 폰트 자동 폴백
- **History API 라우터**: 모든 화면 전환(문서 전환 포함)이 `pushState`로 스택에 기록되어 뒤로가기 시 단계별 복원
- **문서 전환 후 뒤로가기**: 문서(`doc`) 컨텍스트도 history state에 포함되어 정확히 복원
